# git-vue-mall
食用方式
1> cnpm install --save-dev
2> npm run start

编写意义
  梳理vue开发知识点，搭建一个小而不简的多页面应用。并在实践的过程中巩固学习的知识点，对常见的需求进行熟悉和思考。

项目大纲
> src
  > main.js：路由和全局方法挂载。
  > app.vue：渲染固定的header和footer，组件使用</router-link>切换。
  > page
    > detail：详情页相关组件
    > index：首页组件
    > orderList： 列表页相关组件
  > components：其他公共组件
  > assets：图片等静态资源
